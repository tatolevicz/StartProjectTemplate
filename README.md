# StartProjectTemplate
<p align="left"> 
 A handy template to start coding (front/back)end Pixi.js applications in TypeScript building with Webpack.
</p>

</strong>

 <p align="left">
 <strong>2</strong> - Clone this project and run <strong>'npm install' </strong>
